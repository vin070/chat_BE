## Dependency required
1. NodeJS version = v16.20.2
2. npm version = 8.19.4
3. Create a .env file at project root path with below content. You can change configuration according to your setup.
```
# server PORT
PORT = 8000

# database credentials/keys
DB_PASSWORD = '******'
DB_USER = '********'
DB_PORT = 5432
DB_HOST = '0.0.0.0'
DB_NAME = 'chat_app'
```


## Starting project
```
npm run start
```

### Testing project
```
npm run test
```
